# ビデオ喉頭鏡プレテスト｜暗記アプリ

ビデオ喉頭鏡（AWS）追加講習・運用試験の事前学習プレテスト（全63問）を、
スマホで**記憶に残るように暗記できる**学習アプリにしたものです。文字入力は不要、タップだけで使えます。

「眺めるだけ」ではなく、**思い出す練習（テスト効果）＋間隔をあけた復習（間隔反復）**で定着させます。

## 公開URL（スマホで開く）

https://paramedic119.github.io/video-laryngoscope-pretest/

ホーム画面に追加すると、アプリのように起動でき、電波の無い場所でもオフラインで使えます（PWA）。

## 3つのモード

- **暗記**… 全63問を一覧で読む。グレーの □ をタップで答え表示。「答えを表示」で全問まとめて表示。
- **思い出す**（おすすめ）… 1問ずつ「頭の中で答えてから」**答え合わせ**。
  - 答え合わせ後に **○覚えてた／△あいまい／✕忘れた** を1タップで自己採点。
  - 採点結果はスマホに記録され、**○は間隔をあけて、✕は翌日にまた出題**されます（間隔反復）。
  - フィルタ：**今日の復習／すべて／苦手／未学習** をチップで切り替え。
- **力試し**… ランダム10問のミニテスト。最後に **●/10問・○点** を表示。
  - 結果画面の **「間違えた○問だけ復習」** から、そのまま弱点復習へ。

## 進捗の見える化

画面上部に **定着度バー・マスター/学習中/未学習の数・連続学習日数・今日の問数・今日の復習数** を表示。
すべて端末内（localStorage）に保存されるので**ログイン不要**。`⟲` ボタンで学習記録をリセットできます。

## 難易度（隠す量）

**初級／中級／上級** で虫食いの量が変わります（上級ほど多く隠れる）。暗記・思い出すの両モードで有効。

- **初級**… 各文でいちばん大事な用語だけを隠します
- **中級**… 専門用語・主要な数字も隠します
- **上級**… 漢字・数字をほぼすべて隠します（動詞・助詞・ひらがなは残ります）

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | 画面の骨組み（進捗パネル・モード切替・3つのビュー） |
| `styles.css` | デザイン（Apple風・モバイル優先・ダークモード対応） |
| `questions.js` | 全63問のデータ（虫食い箇所を難易度別に仕分け済み）※本体は変更しない |
| `srs.js` | 間隔反復（ライトナー5箱）の計算だけ。画面・保存に依存しない純粋ロジック |
| `storage.js` | 学習記録の保存（localStorage）。問題IDをキーに別管理 |
| `app.js` | 表示・タップ・モード/難易度/フィルタ切替・採点の動作 |
| `manifest.webmanifest`／`sw.js` | PWA（ホーム画面追加・オフライン）。アプリ更新時は `sw.js` の `CACHE` を v2, v3 … と上げる |
| `icon-*.png`／`apple-touch-icon.png`／`favicon-32.png` | アプリアイコン |

ビルド不要・依存ライブラリなしの静的サイトです。`index.html` をブラウザで開くだけでも動きます
（※ Service Worker を使うため、配信は `http(s)://` で。`file://` 直開きではオフライン機能のみ無効）。

## データ設計メモ（拡張する人向け）

- 学習状態は `localStorage` の `vlr_pretest_v1` に保存。`{ version, cards:{ "問ID":{box,due,right,wrong,seen,last} }, streak, day }`。
- `box` 0=未学習／1〜5。○で1つ上（最大5）、△は据え置き翌日、✕は箱1へ。`box>=4` を「マスター」と表示。
- 将来クラウド保存（Firebase）にする場合も、差し替えは `storage.js` の中だけで済む設計です。
