# ビデオ喉頭鏡プレテスト｜虫食い暗記

ビデオ喉頭鏡（AWS）追加講習・運用試験の事前学習プレテスト（全63問）を、
スマホで暗記できる「虫食い問題」アプリにしたものです。文字入力は不要、タップだけで使えます。

## 公開URL（スマホで開く）

https://paramedic119.github.io/video-laryngoscope-pretest/

※ GitHub Pages を有効にすると上記URLで開けます。

## 使い方

- 画面上部の **初級／中級／上級** で難易度を切り替えます。上級ほど隠れる量が増えます。
  - **初級**… 各文でいちばん大事な用語だけを隠します
  - **中級**… 専門用語・主要な数字も隠します
  - **上級**… 漢字・数字をほぼすべて隠します（「連なる」「呼ぶ」などの動詞・助詞・ひらがなは残ります）
- グレーの **□** をタップすると答えが出ます。もう一度タップで隠せます。
- **答えを表示** を押すと全問の答えをまとめて表示、もう一度押すと全部隠します。
- スマホ最適化・ダークモード対応です。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | 画面の骨組み |
| `styles.css` | デザイン（Apple風・モバイル優先・ダークモード対応） |
| `questions.js` | 全63問のデータ（虫食い箇所を難易度別に仕分け済み） |
| `app.js` | 表示・タップ・難易度切替の動作 |

ビルド不要・依存ライブラリなしの静的サイトです。`index.html` をブラウザで開くだけでも動きます。
